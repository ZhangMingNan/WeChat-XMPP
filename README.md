# WeChat-XMPP
基于XMPP实现微信部分功能

![(logo)](http://7u2o6d.com1.z0.glb.clouddn.com/Simulator%20Screen%20Shot%202015年10月24日%20下午8.41.17.png)

技术选型:

*XMPP通讯库  'XMPPFramework'
*HTTP网络库  'Alamofire'
*图片缓存  'Kingfisher'
*状态提示  'MBProgressHUD'
*JSON解析  'SwiftyJSON'
*下拉刷新  'MJRefresh'
*图片上传  'UpYunSDK'
*XPath解析  'hpple'
